Keywords: hotkeys, keyboard shortcut
0. Run:
INTELLIJ
	Ctrl + Shift + F10
	Ctrl + F10
ECLIPSE
	Ctrl + Shift + X
	Double click on [Function Name] -> Run
1. Copy/paste entire line
	Ctrl - Alt - Up		copies current line or selected lines to below
	Ctrl - Alt - Down	copies current line or selected lines to above
	Ctrl - Shift - L	brings up a List of shortcut keys
	See Windows/Preference->General->Keys.

2. Reformat coding / auto-format code
INTELLIJ
	Ctrl + Alt + L
ECLIPSE
	Ctrl + Shift + F	On Windows
	CMD + Up + F		On Mac
	
3. Delete
	Ctrl + D 		Delete current line
	Ctrl + Delete 		Delete next word after cursor
	Ctrl + Shift + Delete	Delete from the cursor until end of line
	
4. Lower + Upper
	Ctrl + Shift + y	Changes a selection to lowercase
	Ctrl + Shift + x	Changes a selection to uppercase
	
5. Line
	Alt + Up 		Moves up current line
	Alt + Control + Up	Copies and moves up current line
	Alt + Down 		Moves up current line
	Alt + Control + Down	Copies and moves down current line
	
6. Import
	Ctrl + Shift + O	Remove unused imports
	Ctrl + Shift + M	Add a single import
				<=> Alt + Enter (Intellij)
							
7. Commect <-> Code
	Ctrl + /		Code to comment
	Ctrl + \		Comment to code
	Ctrl + Shift + /	XML code to comment
	Ctrl + Shift + \	XML commect to code

8. System.out.println
INTELLIJ
	sout + Tab
ECLIPSE
	syso + Ctrl + Space
